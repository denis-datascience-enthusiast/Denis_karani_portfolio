Project I: Heart Disease Prediction Model

April 9, 2023

I have successfully developed and implemented a heart disease prediction model using Logistic Regression machine learning algorithm to determine whether someone has a heart disease or not. My responsibilities included:

  • Dataset collection – heart disease dataset is collected using secondary means of data collection.
  • Importing Libraries: NumPy, Pandas, Scikit-learn, Matplotlib and Seaborn libraries are to be used.
  • Exploratory data analysis: For getting more insights about data.
  • Data cleaning and preprocessing: Check for null and junk values using isnull() and isna().sum() functions of python. In Preprocessing phase, feature engineering on our dataset. convert categorical variables into numerical variables using get_dummies () function of Pandas library. Both our datasets contain some categorical variables
  • Feature Scaling: In this step, we normalize our data by applying Standardization by using StandardScalar() and fit_transform() functions of scikitlearn library.
  • splitting test and training data: using by the importing model selection function of the sklearn library, data is to be splitted into X’s and Y’s of train and test split using train_test_split() function of sklearn. It should be splitted 80% of our data for training and 20% for testing.
  • Apply Logistic Regression Machine Learning model and create a confusion matrix for the model.
  • Deployment of the model.
